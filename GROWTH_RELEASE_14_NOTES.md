# Growth Release 14 — SEO Intelligence Engine

This release improves every public rule page without creating thin pages.

## Included
- Search-intent titles for airline-specific and general item rules.
- Concise meta descriptions containing cabin and checked-baggage decisions.
- Improved category → item → airline breadcrumb hierarchy.
- WebPage structured data with modification date, topic and keywords.
- A visible “Rule at a glance” facts panel.
- Cross-airline links for the same item, where supporting rule data exists.
- Build-time validation for all Growth Release 14 integration points and duplicate rule slugs.

## Release check
Run `npm run build`. The final validation should report:

`Validated Growth Release 14 SEO Intelligence Engine across … rule pages.`
