# Release 6 — Travel Intelligence Graph

This release upgrades the existing graph foundation into a weighted relationship engine.

## Included
- Graph schema 2.0 with typed weighted edges.
- Rule-to-item, category, airline, country and source relationships.
- Rule-to-rule similarity based on shared item, category, baggage outcome and topic tags.
- Graph-ranked connections on every rule page.
- Journey graph coverage score and recommendations inside Trip Checker.
- Local graph report and post-build validation.

## Safety decisions
- No new public URL explosion.
- No automatic publication of combinations.
- Editorial sources remain clearly labelled.
- Missing official-source coverage is shown rather than concealed.
- Recommendations are based on measurable data relationships.
