# Version 2.1 — Authority & Traffic Foundation

## Added
- `/guides/` Travel Knowledge Centre hub.
- Eight substantial evergreen travel guides.
- Dynamic, statically exported guide template with Article and Breadcrumb structured data.
- Contextual Knowledge Centre links on every rule page.
- Related rule and guide navigation.
- Guides link in the global footer.
- Post-build validation for guide exports, sitemap coverage, footer navigation and rule integration.

## Build checks
Run `npm run build`. A successful build ends with a message confirming that all knowledge guides and sitemap URLs were validated.
