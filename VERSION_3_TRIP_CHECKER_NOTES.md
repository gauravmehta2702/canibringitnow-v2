# Version 3.0 — Trip Packing & Baggage Rule Checker

Adds a new public tool at `/trip-checker/` that lets visitors:

- select an airline and destination;
- choose cabin or checked baggage;
- add multiple items from the travel-rule database;
- receive allowed, restricted and not-allowed totals;
- open the detailed rule for every selected item;
- copy or print a trip summary.

The release also adds homepage and footer links, WebApplication structured data, sitemap inclusion, and a post-build validator.
