# Version 4.0 — Travel Intelligence Engine

This release introduces the first unified journey-intelligence layer used by the Trip Checker.

## Included
- Persistent departure country and traveller profile
- Unified `TravelProfile` data model
- Context-aware findings for airline, destination, bag and item combinations
- Readiness scoring with clear status labels
- Journey-stage checklist: home, security, boarding and arrival
- Improved copied/shared trip report
- Build-time integration validation

## Test
Run `npm run build`, then `npm run dev` and open `/trip-checker/`.
